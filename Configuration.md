# Introduction #

To setup Moveable Weather, you'll have to set up the following major components:

  1. [Download and customize the source files.](DownloadCustomize.md)
  1. [Create and setup  new Tropo app.](TropoSetup.md)
  1. [Register the domain of your new App Engine app.](RegisterDomain.md)
  1. [Bulk load zip code data.](BulkLoad.md)
  1. [For Moveable Tides, bulk load tide station data.](BulkLoadTideStations.md)
  1. [For Moveable History, bulk load NRHP data.](BulkLoadNRHP.md)
